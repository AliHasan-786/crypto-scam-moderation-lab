# Reviewer Study Kit

This is an informal usability/quality-feedback kit, not human-subjects research
approval. Ali must confirm the intended activity and any institutional-review
requirements before recruiting. It supports a 5–8 person study using sanitized
authored cases and aggregate-only reporting.

```bash
python3 quality/study_kit/build_packet.py
python3 quality/study_kit/reviewer_study_metrics.py --responses responses.csv --out aggregate.json
```
